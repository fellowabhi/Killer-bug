# VS Code Debug MCP Server

AI-powered debugging for VS Code via Model Context Protocol.

## Installation

1. Install from VS Code marketplace
2. Extension will auto-configure MCP on first use

## Usage

The extension runs automatically when debugging is active. AI assistants can interact with the debugger through MCP tools.

## Development

```bash
npm install
npm run build
# Press F5 in VS Code to launch extension development host
```
